SLSC EDS Addon - Commands

This SLSC EDS Addon provides the ability to monitor and communicate with SLSC Module inside a SLSC Chassis. The Command Parameters are used to activate "Commands" available on your SLSC Module through the this Addon.

What do you want to do?

Make a connection to the Command Channels

Connect to the Command channel if you want to active the command by applying any value different than zero. When the command is activated, this channel will be changed back to zero.
